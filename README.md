## 📘 Descripción general del sistema
LlanquihueTourApp es un sistema desarrollado en Java para represengtar y gestionar tour turisticos de la agencia Llanquihue tour
La aplicación aplica principios de POO, lectura de archivos, uso de colecciones y composición entre clases
permitiendo cargar, visualizar y filtrar tours según precio.


## Conceptos Aplicados
- Programación Orientada a Objetos.
- Encapsulamiento mediante atributos privados.
- Uso de constructores, getters y setters.
- Sobrescritura del método `toString()`.
- Composición entre clases.
- Organización modular del proyecto en paquetes.
- Lectura de archivos .txt.
- Separación de datos mediante `split(";")`.
- Uso de colección ArrayList.
- Recorrido de colecciones con bucle `for-each`.
- Filtrado de objetos según una condición.
- Manejo básico de errores con try-catch.
